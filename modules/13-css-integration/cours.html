Intégration CSS

Liez CSS à votre HTML de manière professionnelle. Comprenez les trois méthodes d'intégration, organisez votre HTML pour le CSS, et maîtrisez les sélecteurs, les classes et les identifiants.

13.1 — Les trois méthodes d'intégration CSS

MéthodeSyntaxeRecommandée ?Usage
CSS externe<link rel="stylesheet" href="style.css">✅ ToujoursProduction — fichier séparé réutilisable
CSS interne<style>...</style> dans le <head>⚠️ ParfoisEmails HTML, prototypes rapides
CSS en lignestyle="..." sur une balise❌ ÉviterSeulement si généré dynamiquement
CSS externe — méthode recommandéeHTML
<!-- Dans le <head> -->

<!-- Feuille de style principale -->
<link rel="stylesheet" href="css/style.css">

<!-- Police Google Fonts -->
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="stylesheet"
      href="https://fonts.googleapis.com/css2?family=Inter:wght@400;700&display=swap">

<!-- CSS conditionnel (impression) -->
<link rel="stylesheet" href="css/print.css" media="print">

13.2 — Classes et identifiants : organiser le HTML pour le CSS

Utilisation des classes et IDsHTML
<!-- class : peut être utilisée sur plusieurs éléments -->
<p class="introduction">Premier paragraphe intro.</p>
<p class="introduction">Deuxième paragraphe intro.</p>

<!-- Plusieurs classes sur un même élément -->
<button class="bouton bouton-primaire bouton-large">S'inscrire</button>

<!-- id : unique dans la page — pour JS et ancres -->
<section id="contact">
  <!-- ciblé par href="#contact" -->
</section>

<!-- Attribut data-* pour le JavaScript -->
<button
  class="btn-produit"
  data-id="42"
  data-prix="29.99"
  data-nom="Cours HTML"
>
  Ajouter au panier
</button>

13.3 — Variables CSS et structure HTML moderne

Fichier CSS avec variables (design tokens)CSS
/* Variables CSS globales — définies dans :root */
:root {
  --couleur-primaire:    #3b82f6;
  --couleur-texte:       #1f2937;
  --couleur-fond:        #ffffff;
  --police-principale:   'Inter', sans-serif;
  --taille-base:         1rem;
  --espacement-md:       1rem;
  --rayon-bordure:       8px;
}

/* Utilisation des variables */
body {
  font-family: var(--police-principale);
  color: var(--couleur-texte);
  background: var(--couleur-fond);
}

.bouton-primaire {
  background: var(--couleur-primaire);
  border-radius: var(--rayon-bordure);
  padding: var(--espacement-md);
}
✏️ Exercice 13.1 — Styliser une page complète

Reprenez votre page du module 10 (HTML sémantique) et créez un fichier style.css séparé. Liez-le avec <link>. Ajoutez des classes à vos éléments HTML. Stylez : couleur de fond du body, typographie (font-family, color, font-size), header avec couleur de fond, liens avec couleur personnalisée et suppression du soulignement. Testez sur mobile (DevTools → mode responsive).
