Intégration JavaScript

Intégrez JavaScript dans vos pages HTML. Comprenez les différentes façons de lier du JS, la manipulation du DOM, les événements, et les bonnes pratiques de chargement.

14.1 — La balise <script>

La balise <script> permet d'inclure ou de référencer du JavaScript dans une page HTML.

Intégration JavaScriptHTML
<!-- ✅ Méthode recommandée : script externe, en bas du body -->
<body>
  <!-- ... tout le HTML ... -->
  <script src="js/script.js"></script>
</body>

<!-- Script avec defer : télécharge en parallèle, s'exécute après le HTML -->
<head>
  <script src="js/script.js" defer></script>
</head>

<!-- Script avec async : télécharge et s'exécute dès que possible -->
<script src="js/analytics.js" async></script>

<!-- Script inline (pour petits bouts de code seulement) -->
<script>
  console.log('Page chargée !');
</script>

14.2 — Introduction au DOM

Le DOM (Document Object Model) est une représentation en mémoire de votre page HTML sous forme d'arbre d'objets. JavaScript peut lire et modifier le DOM pour changer dynamiquement le contenu et l'apparence de la page.

Manipulation du DOM — les basesJavaScript
// Sélectionner des éléments HTML
const titre = document.getElementById('mon-titre');
const boutons = document.querySelectorAll('.bouton');
const premier = document.querySelector('h1');

// Modifier le contenu
titre.textContent = 'Nouveau titre !';
titre.innerHTML = 'Titre avec <strong>gras</strong>';

// Modifier les attributs
document.querySelector('img').setAttribute('alt', 'Nouveau alt');

// Ajouter/supprimer des classes
titre.classList.add('actif');
titre.classList.remove('cache');
titre.classList.toggle('visible');

14.3 — Les événements JavaScript

Gestion des événementsHTML + JavaScript
<!-- HTML : bouton avec id -->
<button type="button" id="mon-bouton">Cliquez-moi</button>
<p id="resultat">...</p>

/* JavaScript */
const bouton = document.getElementById('mon-bouton');
const resultat = document.getElementById('resultat');

bouton.addEventListener('click', function() {
  resultat.textContent = 'Bouton cliqué !';
});

/* Événements courants */
element.addEventListener('click', handler);      // clic souris
element.addEventListener('keydown', handler);    // touche pressée
element.addEventListener('submit', handler);     // formulaire envoyé
element.addEventListener('input', handler);      // saisie dans un champ
element.addEventListener('mouseover', handler);  // survol souris
window.addEventListener('scroll', handler);     // défilement page
document.addEventListener('DOMContentLoaded', handler); // DOM prêt
✏️ Exercice 14.1 — Interactivité avec JS

Créez une page avec un bouton "Mode sombre" qui alterne entre un fond blanc et un fond noir en modifiant la classe du <body>. En bonus : sauvegardez la préférence dans localStorage et restaurez-la au chargement.
