Tableaux HTML

Créez des tableaux HTML structurés et accessibles. Maîtrisez thead, tbody, tfoot, colspan et rowspan pour des tableaux complexes. Comprenez quand utiliser (et ne pas utiliser) les tableaux.

8.1 — Structure d'un tableau HTML

Un tableau HTML est composé de plusieurs balises imbriquées. Voici la structure complète :

BaliseSignificationRôle
<table>TableConteneur du tableau complet
<caption>CaptionTitre/légende du tableau (accessibilité)
<thead>Table HeadGroupe de lignes d'en-tête
<tbody>Table BodyGroupe de lignes de données
<tfoot>Table FootGroupe de lignes de pied (totaux, résumés)
<tr>Table RowUne ligne du tableau
<th>Table HeaderCellule d'en-tête (gras, centré, scope)
<td>Table DataCellule de données normale
Tableau HTML complet et accessibleHTML
<table>
  <!-- Titre du tableau (recommandé pour l'accessibilité) -->
  <caption>Résultats des étudiants — Semestre 1 2024</caption>

  <!-- En-tête : contient les titres de colonnes -->
  <thead>
    <tr>
      <th scope="col">Nom</th>
      <th scope="col">HTML</th>
      <th scope="col">CSS</th>
      <th scope="col">Moyenne</th>
    </tr>
  </thead>

  <!-- Corps : contient les données -->
  <tbody>
    <tr>
      <th scope="row">Alice</th>
      <td>18/20</td>
      <td>16/20</td>
      <td>17/20</td>
    </tr>
    <tr>
      <th scope="row">Bob</th>
      <td>14/20</td>
      <td>19/20</td>
      <td>16.5/20</td>
    </tr>
  </tbody>

  <!-- Pied de tableau : totaux, moyennes globales -->
  <tfoot>
    <tr>
      <th scope="row">Moyenne de la classe</th>
      <td>16/20</td>
      <td>17.5/20</td>
      <td>16.75/20</td>
    </tr>
  </tfoot>
</table>

8.2 — colspan et rowspan : fusionner des cellules

colspan fusionne des cellules horizontalement (sur plusieurs colonnes). rowspan fusionne des cellules verticalement (sur plusieurs lignes).

Fusion de cellulesHTML
<table>
  <thead>
    <tr>
      <!-- Cette cellule occupe 2 colonnes -->
      <th colspan="2">Informations personnelles</th>
      <th colspan="2">Notes</th>
    </tr>
    <tr>
      <th>Nom</th><th>Prénom</th>
      <th>HTML</th><th>CSS</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <!-- rowspan : cette cellule couvre 2 lignes -->
      <td rowspan="2">Dupont</td>
      <td>Alice</td>
      <td>18</td><td>16</td>
    </tr>
    <tr>
      <!-- Pas de <td> pour "Dupont" ici — rowspan s'en charge -->
      <td>Bernard</td>
      <td>14</td><td>19</td>
    </tr>
  </tbody>
</table>
⚠️ Tableaux : usage sémantique uniquement

Les tableaux HTML doivent être utilisés uniquement pour des données tabulaires (données qui ont du sens dans un tableau à lignes et colonnes). Ne jamais utiliser les tableaux pour la mise en page — c'est une pratique obsolète des années 1990. La mise en page est le rôle du CSS (Flexbox, Grid).

✏️ Exercice 8.1 — Tableau de comparaison

Créez un tableau comparant 3 forfaits téléphoniques (Basic, Pro, Premium). Le tableau doit avoir : une caption, un thead avec les noms des forfaits, un tbody avec 5 caractéristiques (Prix, Appels, SMS, Data, Options), et un tfoot avec un bouton "Choisir". Utilisez colspan pour le titre principal.
