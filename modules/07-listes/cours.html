Listes HTML

Structurez vos données avec les listes HTML : listes non ordonnées, ordonnées et de définitions. Apprenez à créer des listes imbriquées et à les utiliser pour la navigation.

7.1 — Listes non ordonnées <ul>

Une liste non ordonnée (<ul> = unordered list) affiche des éléments avec des puces. L'ordre des éléments n'a pas d'importance sémantique. Chaque élément est encadré par une balise <li> (list item).

Liste non ordonnéeHTML
<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

7.2 — Listes ordonnées <ol>

Une liste ordonnée (<ol> = ordered list) numérote ses éléments automatiquement. À utiliser quand l'ordre des éléments est important (étapes, classements, instructions).

Liste ordonnée et ses attributsHTML
<!-- Liste ordonnée classique (commence à 1) -->
<ol>
  <li>Ouvrir VS Code</li>
  <li>Créer un fichier index.html</li>
  <li>Taper le DOCTYPE</li>
</ol>

<!-- Démarrer à un numéro précis -->
<ol start="5">
  <li>Cinquième étape</li>
  <li>Sixième étape</li>
</ol>

<!-- Compte à rebours -->
<ol reversed>
  <li>Top 3 : Bronze</li>
  <li>Top 2 : Argent</li>
  <li>Top 1 : Or</li>
</ol>

<!-- Numérotation avec des lettres -->
<ol type="A">  <!-- A, B, C... -->
  <li>Option A</li>
  <li>Option B</li>
</ol>

<!-- Chiffres romains -->
<ol type="I">  <!-- I, II, III... -->
  <li>Chapitre I</li>
  <li>Chapitre II</li>
</ol>

7.3 — Listes imbriquées

Il est possible d'imbriquer des listes à l'intérieur d'éléments <li> pour créer des structures hiérarchiques (menus, plans, organigrammes).

Listes imbriquées — menu de navigationHTML
<ul>
  <li>Développement Web
    <ul>
      <li>HTML</li>
      <li>CSS
        <ul>
          <li>Flexbox</li>
          <li>Grid</li>
        </ul>
      </li>
      <li>JavaScript</li>
    </ul>
  </li>
  <li>Design
    <ul>
      <li>Figma</li>
      <li>Photoshop</li>
    </ul>
  </li>
</ul>

7.4 — Listes de définitions <dl>

La liste de définitions est moins connue mais très utile pour les paires terme-définition : glossaires, FAQ, métadonnées d'un article, dictionnaires.

  • <dl> : definition list — le conteneur
  • <dt> : definition term — le terme à définir
  • <dd> : definition description — la définition
Liste de définitionsHTML
<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language. Langage de balisage pour structurer les pages Web.</dd>

  <dt>CSS</dt>
  <dd>Cascading Style Sheets. Langage de mise en forme des pages Web.</dd>

  <dt>JavaScript</dt>
  <dd>Langage de programmation qui rend les pages Web interactives.</dd>

  <!-- Un terme peut avoir plusieurs définitions -->
  <dt>Web</dt>
  <dd>World Wide Web — réseau mondial de pages reliées par des liens.</dd>
  <dd>Toile d'araignée — d'où vient la métaphore du Web.</dd>
</dl>
✏️ Exercice 7.1 — Menu de navigation avec listes

Créez la navigation d'un site e-commerce avec une liste imbriquée : Catégories (Hommes, Femmes, Enfants — chacune avec 3 sous-catégories), Services (Livraison, Retours, FAQ), et Contact. Chaque élément doit être un lien avec href="#".
