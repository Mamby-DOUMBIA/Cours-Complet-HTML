HTML Avancé

Explorez les fonctionnalités avancées du HTML : attributs data-*, Web Components, microdata, iframes, et les éléments HTML5 moins connus mais très puissants.

15.1 — Attributs data-* personnalisés

Les attributs data-* permettent de stocker des données personnalisées directement dans les éléments HTML. Ils sont lus par JavaScript sans modifier la structure ou le style de la page.

Attributs data-* en pratiqueHTML + JavaScript
<!-- Stocker des données sur les produits d'une boutique -->
<div
  class="carte-produit"
  data-id="prod-42"
  data-prix="29.99"
  data-stock="5"
  data-categorie="electronique"
>
  <h3>Casque audio premium</h3>
  <p>29,99 €</p>
  <button type="button">Ajouter au panier</button>
</div>

/* Lire les data attributes en JavaScript */
const carte = document.querySelector('.carte-produit');
const id    = carte.dataset.id;        // "prod-42"
const prix  = carte.dataset.prix;      // "29.99"
const stock = carte.dataset.stock;     // "5"
console.log(`Produit ${id} à ${prix} €`);

15.2 — L'élément <dialog> natif

Modal native HTML avec dialogHTML + JavaScript
<button type="button" id="ouvrir-modal">Ouvrir la modal</button>

<dialog id="ma-modal" aria-labelledby="titre-modal">
  <h2 id="titre-modal">Confirmation</h2>
  <p>Voulez-vous vraiment supprimer cet élément ?</p>
  <menu>
    <button type="button" id="annuler">Annuler</button>
    <button type="button" id="confirmer">Confirmer</button>
  </menu>
</dialog>

<script>
  const modal = document.getElementById('ma-modal');
  document.getElementById('ouvrir-modal')
    .addEventListener('click', () => modal.showModal());
  document.getElementById('annuler')
    .addEventListener('click', () => modal.close());
</script>

15.3 — <template> : contenu HTML non rendu

Balise templateHTML + JavaScript
<!-- template : son contenu n'est pas affiché, mais cloneable par JS -->
<template id="carte-template">
  <article class="carte">
    <h3 class="carte-titre"></h3>
    <p class="carte-description"></p>
  </article>
</template>

<div id="conteneur"></div>

<script>
  const tmpl = document.getElementById('carte-template');
  const clone = tmpl.content.cloneNode(true);
  clone.querySelector('.carte-titre').textContent = 'Titre dynamique';
  clone.querySelector('.carte-description').textContent = 'Description...';
  document.getElementById('conteneur').appendChild(clone);
</script>
