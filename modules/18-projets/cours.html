Projets Pratiques Complets

Appliquez tout ce que vous avez appris en construisant 5 projets HTML complets et professionnels, de la page personnelle au site e-commerce.

Les 5 projets du cours

Ces projets sont conçus pour consolider toutes vos connaissances en HTML. Chaque projet est plus complexe que le précédent et introduit de nouveaux défis.

Projet 1 — Page personnelle

Objectif : Créer une page de présentation personnelle complète et professionnelle.

Ce que vous allez pratiquer : Structure HTML5, sémantique, images, liens, formulaire de contact, accessibilité.

Structure cible — index.htmlHTML
<body>
  <header>
    <nav>Navigation : Accueil | À propos | Compétences | Projets | Contact</nav>
  </header>

  <main>
    <!-- Section Hero -->
    <section id="accueil">
      <img src="photo.jpg" alt="[Votre nom], développeur Web">
      <h1>Prénom NOM</h1>
      <p>Développeur Web spécialisé en HTML & CSS</p>
    </section>

    <!-- Section À propos -->
    <section id="a-propos">
      <h2>À propos de moi</h2>
      <p>Votre description...</p>
    </section>

    <!-- Section Compétences -->
    <section id="competences">
      <h2>Mes compétences</h2>
      <ul>
        <li>HTML5</li>
        <li>CSS3</li>
        <li>JavaScript</li>
      </ul>
    </section>

    <!-- Section Contact -->
    <section id="contact">
      <h2>Contactez-moi</h2>
      <form action="#" method="post">
        <!-- Champs : Nom, Email, Message, Bouton Envoyer -->
      </form>
    </section>
  </main>

  <footer>
    <p>&copy; 2024 Prénom NOM — Tous droits réservés</p>
  </footer>
</body>

Projet 2 — Site multipage

Objectif : Construire un site Web de 5 pages avec navigation cohérente.

Pages à créer : index.html, a-propos.html, services.html, blog.html, contact.html

Ce que vous allez pratiquer : Navigation entre pages, chemins relatifs, structure commune header/footer, balises sémantiques.

Projet 3 — Page de blog

Objectif : Créer une liste d'articles de blog et une page article complète.

Ce que vous allez pratiquer : <article>, <time>, <figure>, <blockquote>, données structurées JSON-LD.

Projet 4 — Page produit e-commerce

Objectif : Créer une page produit complète avec galerie, description, prix et formulaire d'achat.

Ce que vous allez pratiquer : <picture>, tableaux de caractéristiques, data-attributes, Schema.org Product.

Projet 5 — Formulaire de contact avancé

Objectif : Créer le formulaire de contact le plus complet et accessible possible.

Ce que vous allez pratiquer : Tous les types d'input, validation HTML5, <fieldset>/<legend>, attributs ARIA, messages d'erreur accessibles.

🎉 Félicitations !

Si vous avez complété tous ces projets, vous maîtrisez HTML de A à Z. Vous êtes prêt(e) pour :

  • Apprendre le CSS en profondeur (MDN)
  • Maîtriser JavaScript (MDN)
  • Découvrir un framework front-end (React, Vue, Angular)
  • Construire de vrais projets et les déployer en ligne
