HTML et SEO

Optimisez vos pages HTML pour les moteurs de recherche. Maîtrisez les balises meta essentielles, Open Graph, les données structurées, et toutes les bonnes pratiques SEO on-page.

12.1 — Les balises meta essentielles pour le SEO

Head complet optimisé SEOHTML
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">

  <!-- SEO de base -->
  <title>Guide HTML complet pour débutants — LearnCode.fr</title>
  <meta name="description"
        content="Apprenez le HTML de zéro. 18 modules progressifs, 100+ exemples pratiques. Formation gratuite en français.">
  <meta name="author" content="Jean Dupont">
  <meta name="robots" content="index, follow">
  <link rel="canonical" href="https://learncode.fr/html/guide-complet">

  <!-- Open Graph (Facebook, LinkedIn, WhatsApp...) -->
  <meta property="og:type"        content="article">
  <meta property="og:title"       content="Guide HTML complet pour débutants">
  <meta property="og:description"  content="Apprenez le HTML de zéro à expert.">
  <meta property="og:image"       content="https://learncode.fr/images/og-html.jpg">
  <meta property="og:url"         content="https://learncode.fr/html/guide-complet">
  <meta property="og:locale"      content="fr_FR">
  <meta property="og:site_name"   content="LearnCode.fr">

  <!-- Twitter Card -->
  <meta name="twitter:card"        content="summary_large_image">
  <meta name="twitter:title"       content="Guide HTML complet pour débutants">
  <meta name="twitter:description"  content="Apprenez le HTML de zéro à expert.">
  <meta name="twitter:image"       content="https://learncode.fr/images/og-html.jpg">

  <!-- Favicon -->
  <link rel="icon" type="image/svg+xml" href="/favicon.svg">
  <link rel="apple-touch-icon" href="/apple-touch-icon.png">
</head>

12.2 — Données structurées (Schema.org)

Les données structurées permettent à Google d'afficher des rich snippets dans ses résultats (étoiles, prix, disponibilité, FAQ…). On les ajoute en JSON-LD dans un <script type="application/ld+json">.

Données structurées JSON-LD — ArticleHTML
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Article",
  "headline": "Guide HTML complet pour débutants",
  "author": {
    "@type": "Person",
    "name": "Jean Dupont"
  },
  "datePublished": "2024-03-15",
  "dateModified": "2024-06-01",
  "image": "https://learncode.fr/images/guide-html.jpg",
  "publisher": {
    "@type": "Organization",
    "name": "LearnCode.fr",
    "logo": {
      "@type": "ImageObject",
      "url": "https://learncode.fr/logo.svg"
    }
  }
}
</script>

12.3 — Checklist SEO HTML complète

✅ Checklist SEO on-page
  • Title unique par page, 50–60 caractères, mot-clé principal en tête
  • Meta description unique par page, 150–160 caractères, incitative
  • Un seul h1 par page, contenant le mot-clé principal
  • Hiérarchie des titres logique (h1 → h2 → h3)
  • Attribut alt sur toutes les images, avec description incluant le contexte
  • URL descriptives et lisibles (pas de paramètres cryptiques)
  • Balise canonical pour éviter le contenu dupliqué
  • Open Graph pour les partages sur réseaux sociaux
  • Données structurées JSON-LD pour les rich snippets
  • lang="fr" sur la balise html
  • Liens internes avec ancres textuelles descriptives
  • Vitesse de chargement optimisée (images compressées, loading="lazy")
