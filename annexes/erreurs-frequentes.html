📎 Annexe

30 erreurs HTML fréquentes

💡 Comment utiliser cette liste

Chaque erreur est accompagnée d'un exemple incorrect, d'un exemple corrigé et d'une explication. Parcourez cette liste régulièrement pour développer de bons réflexes.

Erreurs de structure (modules 03-04)

❌ Erreur 1 : DOCTYPE manquant
<html lang="fr">
<head>...
✅ Correction
<!DOCTYPE html>
<html lang="fr">

Le DOCTYPE doit être la toute première ligne du fichier, avant même <html>.

❌ Erreur 2 : lang manquant sur html
<html>
✅ Correction
<html lang="fr">
❌ Erreur 3 : charset après title
<head>
  <title>Ma page</title>
  <meta charset="UTF-8">
✅ Correction : charset EN PREMIER
<head>
  <meta charset="UTF-8">
  <title>Ma page</title>
❌ Erreur 4 : plusieurs h1
<h1>Mon site</h1>
...
<h1>Mon article</h1>
✅ Correction : un seul h1
<h1>Mon site</h1>
...
<h2>Mon article</h2>
❌ Erreur 5 : sauter des niveaux de titre
<h1>Titre</h1>
<h3>Sous-titre</h3>
✅ Correction
<h1>Titre</h1>
<h2>Sous-titre</h2>

Erreurs sur les liens (module 05)

❌ Erreur 6 : target="_blank" sans rel
<a href="https://..." target="_blank">Site</a>
✅ Correction
<a href="https://..."
   target="_blank"
   rel="noopener noreferrer">Site</a>
❌ Erreur 7 : texte de lien non descriptif
<a href="rapport.pdf">Cliquez ici</a>
✅ Correction
<a href="rapport.pdf">Télécharger le rapport annuel (PDF)</a>

Erreurs sur les images (module 06)

❌ Erreur 8 : alt manquant
<img src="photo.jpg">
✅ Correction
<img src="photo.jpg" alt="Description précise de l'image">
❌ Erreur 9 : width/height manquants
<img src="hero.jpg" alt="Hero">
✅ Correction (évite le CLS)
<img src="hero.jpg" alt="Hero" width="1200" height="600">

Erreurs de formulaires (module 09)

❌ Erreur 10 : input sans label
Nom : <input type="text" name="nom">
✅ Correction
<label for="nom">Nom</label>
<input type="text" id="nom" name="nom">
❌ Erreur 11 : tableau pour la mise en page
<table>
  <tr>
    <td>Logo</td>
    <td>Menu</td>
  </tr>
</table>
✅ Correction : CSS pour la mise en page
<header style="display:flex; justify-content:space-between">
  <img src="logo.svg" alt="Logo">
  <nav>Menu</nav>
</header>
❌ Erreur 12 : script bloquant dans head
<head>
  <script src="app.js"></script>
</head>
✅ Correction : defer ou bas du body
<head>
  <script src="app.js" defer></script>
</head>
