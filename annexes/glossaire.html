Glossaire HTML

Attribut
Information supplémentaire fournie à une balise HTML. Exemple : href dans <a href="...">. Les attributs sont toujours dans la balise ouvrante.
Balise
Marqueur HTML entouré de chevrons < >. La plupart viennent par paires (ouvrante + fermante) : <p>...</p>. Certaines sont auto-fermantes : <img>.
BEM (Block Element Modifier)
Convention de nommage CSS. Format : .block__element--modifier. Aide à organiser les classes CSS de façon lisible et scalable.
CLS (Cumulative Layout Shift)
Métrique Core Web Vitals mesurant l'instabilité visuelle. Un CLS élevé signifie que des éléments se déplacent pendant le chargement. Causé par des images sans width/height.
DOM (Document Object Model)
Représentation en mémoire d'une page HTML sous forme d'arbre d'objets. JavaScript peut lire et modifier le DOM pour changer dynamiquement la page.
DOCTYPE
Déclaration obligatoire en début de fichier HTML. <!DOCTYPE html> indique au navigateur d'utiliser le standards mode HTML5.
Élément
Unité de base du HTML. Un élément est composé d'une balise ouvrante, d'un contenu et d'une balise fermante. <p>Ceci est un élément</p>.
Entité HTML
Code de remplacement pour les caractères spéciaux en HTML. Exemple : &lt; affiche <, &amp; affiche &.
HTML (HyperText Markup Language)
Langage de balisage standard pour créer des pages Web. HTML structure le contenu. CSS gère la présentation. JavaScript gère l'interactivité.
LCP (Largest Contentful Paint)
Métrique Core Web Vitals mesurant le temps de rendu du plus grand élément visible. Cible : moins de 2,5 secondes.
Métadonnée
Information sur la page HTML, placée dans le <head> et invisible à l'écran. Lue par les navigateurs, les moteurs de recherche et les réseaux sociaux.
Open Graph
Protocole de métadonnées créé par Facebook. Les balises og: contrôlent l'aperçu d'une page lors du partage sur les réseaux sociaux (titre, description, image).
Schema.org
Vocabulaire partagé par Google, Microsoft et Yahoo pour les données structurées. Permet aux moteurs de recherche de comprendre le contenu d'une page en détail (recette, produit, événement…).
SEO (Search Engine Optimization)
Ensemble de techniques pour améliorer la visibilité d'une page dans les résultats des moteurs de recherche. Le HTML joue un rôle crucial via les balises title, meta description, h1, alt, etc.
Sémantique
En HTML, utiliser des balises qui décrivent la signification du contenu, pas son apparence. <strong> pour l'importance, <nav> pour la navigation, <article> pour un article autonome, etc.
URL (Uniform Resource Locator)
Adresse d'une ressource sur Internet. Exemple : https://developer.mozilla.org/fr/docs/Web/HTML.
Validateur W3C
Outil officiel pour vérifier qu'un document HTML respecte les standards. Disponible sur validator.w3.org.
Viewport
Zone visible de la fenêtre du navigateur. La balise <meta name="viewport"> contrôle comment la page s'affiche sur mobile.
WCAG (Web Content Accessibility Guidelines)
Recommandations du W3C pour l'accessibilité du contenu Web. WCAG 2.1 AA est le standard minimal requis dans de nombreux pays.
WHATWG
Web Hypertext Application Technology Working Group. Organisme qui maintient le standard HTML Living Standard, utilisé comme référence principale par les navigateurs.
